# Memory Cleanup Skill

自动整理记忆系统，保持 MEMORY.md 简洁高效。

## 脚本使用方法（可选）

```bash
python3 skills/memory_cleanup/memory_cleanup.py
```

## 整理规则

### ✅ 保留在 MEMORY.md

- 用户偏好（沟通风格、技术偏好等）
- 身份信息（姓名、角色、关系）
- 项目背景（技术栈、架构决策）
- 长期有效的上下文

**只有 3 个问题的答案都是"是"，才记入 MEMORY.md**
在写入前先问自己：

1. 这个信息**下周/下个月**还会用得上吗？
2. 这个信息会影响**未来的决策**吗？
3. 这个信息是**稳定事实**还是**临时状态**？

> **MEMORY.md 是"长期记忆"，不是"日记本"，不是"技术文档"，不是"待办清单"**

- **简洁**: 只记要点，不记流水账
- **稳定**: 只记长期有效的信息
- **可复用**: 只记会影响未来交互的信息
- **去重**: 其他地方有的就不要重复记

### 📝 移到 HISTORY.md

- 时间线条目（`[YYYY-MM-DD HH:MM] ...`）
- 临时状态（"进行中"、"testing"）
- 过程细节（debug 记录、临时笔记）

## 输出示例

```
🧹 开始整理记忆... (2026-03-03 03:00)
✓ MEMORY.md 整理完成：保留 45 行，移动 12 行时间线条目到 HISTORY.md
✨ 记忆整理完成！
```
